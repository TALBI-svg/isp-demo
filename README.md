# ISP Demo

Small TypeScript demo project showing a minimal structure with `src/main.ts`, `src/good.ts`, and `src/bad.ts`.

## Prerequisites

- Node.js (16+)
- npm or yarn

## Install

```bash
npm install
```

## Build & Run

Compile TypeScript and run the project:

```bash
npm run build
node dist/main.js
```

Or run via ts-node (if installed):

```bash
npx ts-node src/main.ts
```

## Project Files

- `src/main.ts` — entry point
- `src/good.ts` — example good code
- `src/bad.ts` — example bad code
